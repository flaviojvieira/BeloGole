# Belo Gole

Landing page para a adega Belo Gole, com foco em promoções, cardápio, contato e localização para pedidos via WhatsApp.

## Estrutura do projeto
- index.html: estrutura principal da página
- style.css: estilos e responsividade
- js/main.js: comportamento da interface
- js/data.js: conteúdo do cardápio, promoções e dados de configuração
- assets/: imagens usadas no site

## Como visualizar localmente
1. Abra a pasta do projeto no navegador.
2. Se preferir, rode um servidor simples, por exemplo:
   - Python: python -m http.server 8000
   - Depois acesse http://localhost:8000
